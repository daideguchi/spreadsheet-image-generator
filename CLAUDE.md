# Google Apps Script 画像生成ツール - Claude Code設定

## プロジェクト概要

OpenAI DALL-E APIを使用してスプレッドシートに画像生成・挿入するGoogle Apps Scriptツールです。

### 主要機能
- 🎨 DALL-E API連携による画像生成
- 📊 9列構造化スプレッドシート
- 🔄 バッチ処理対応
- 🎯 直感的なUI（色分けシステム）
- 📱 サイドバーUI

## 技術スタック

- **プラットフォーム**: Google Apps Script (V8ランタイム)
- **AI API**: OpenAI DALL-E (v1)
- **UI**: HTML/CSS/JavaScript
- **データ**: Google Spreadsheets
- **認証**: OAuth 2.0

### スプレッドシート構造（A-I列）
| 列 | 用途 | 入力方式 | 色分け |
|---|---|---|---|
| A | No. | 自動連番 | グレー |
| B | プロンプト | ユーザー入力 | 緑 |
| C | 共通プロンプト | ドロップダウン | オレンジ |
| D | 結合プロンプト | 自動生成 | グレー |
| E | 生成画像 | 自動挿入 | グレー |
| F | 画像比率 | 自動計算 | グレー |
| G | 生成日時 | 自動記録 | グレー |
| H | ステータス | 自動更新 | グレー |
| I | エラーログ | 自動記録 | グレー |

## 開発ガイドライン

### コーディング規約
- **関数名**: キャメルケース（generateImageFromPrompt）
- **変数名**: 分かりやすい命名（imagePrompt, generatedImageUrl）
- **定数**: 大文字スネークケース（OPENAI_API_ENDPOINT）
- **エラーハンドリング**: try-catch必須
- **ログ**: console.log()で適切な情報出力

### パフォーマンス考慮
- **実行時間制限**: 6分以内でバッチ処理
- **API制限**: レート制限（60req/min）遵守
- **メモリ効率**: 大量データは分割処理
- **キャッシュ**: PropertiesServiceで結果保存

### セキュリティ
- **APIキー**: PropertiesServiceで暗号化保存
- **入力検証**: プロンプトの適切な検証
- **権限**: 最小限のOAuthスコープ
- **エラー情報**: 機密情報の漏洩防止

## API統合

### OpenAI DALL-E設定
```javascript
const OPENAI_CONFIG = {
  endpoint: 'https://api.openai.com/v1/images/generations',
  model: 'dall-e-3',
  size: '1024x1024',
  quality: 'standard',
  n: 1
};
```

### レート制限対応
- 1分間60リクエスト制限
- 指数バックオフでリトライ
- バッチサイズ調整（5件ずつ処理）

## 設定・環境変数

### 必須設定（PropertiesService）
```javascript
const REQUIRED_PROPERTIES = {
  'OPENAI_API_KEY': 'sk-xxx...', // OpenAI APIキー
};
```

### 設定方法
```javascript
function setupEnvironment() {
  const properties = PropertiesService.getScriptProperties();
  properties.setProperty('OPENAI_API_KEY', 'your-api-key-here');
}
```
